# Exploratory Data Analysis (EDA) - Sample Superstore

## Objective
This project analyzes sales data to find insights like top category, profit, and discount impact.

## Questions
- Which category has highest sales?
- Which region has highest profit?
- Does discount affect profit?
- Which sub-category is most profitable?
- Any loss-making products?

## Tools Used
Python, Pandas, Matplotlib

## How to Run
pip install pandas matplotlib
python task2.py
